# sense_msgs
[Sense's](http://senseit.gr) official ROS messages repository. This ROS 1 package contains commonly used messages which ususally find use in UAS and robotics applications.

## Dependencies:
- *std_msgs*
- *sensor_msgs*
- *geometry_msgs*
- *message_generation*
